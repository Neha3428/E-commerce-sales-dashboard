# 📊 E-Commerce Sales & Profit Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on an E-commerce Superstore dataset to uncover business insights related to sales performance, profitability, product categories, and customer segments.

The analysis was conducted using Python in Jupyter Notebook and includes interactive visualizations built with Plotly.


## Objectives

 Analyze monthly sales trends,
 Evaluate profit performance across different periods,
 Identify top-performing product categories and sub-categories,
 Compare sales and profit across customer segments,
 Calculate Sales-to-Profit ratios to assess business efficiency,
 Generate actionable business insights from transactional data.

## Dataset

The project uses the **Sample Superstore Dataset**, which contains information such as:

 Order Date,
 Ship Date,
 Sales,
 Profit,
 Category,
 Sub-Category,
 Segment,
 Customer Information,
 Product Information.

## Project Workflow

### 1. Data Loading & Inspection

 Imported dataset using Pandas,
 Performed initial data exploration,
 Checked data types and dataset structure.

### 2. Data Preprocessing

 Converted date columns into datetime format.
 Extracted:

   Order Month,
   Order Year,
   Day of Week.

### 3. Sales Analysis

 Monthly Sales Analysis,
 Sales by Category,
 Sales by Sub-Category.

### 4. Profit Analysis

 Monthly Profit Analysis,
 Profit by Category,
 Profit by Sub-Category.
  
## Visualizations

The notebook includes:

 Line Charts,
 Bar Charts,
 Pie Charts,
 Comparative Sales vs Profit Charts,
 Segment-wise Performance Analysis.

All visualizations were created using Plotly for interactive data exploration.

## Key Insights

 Identified monthly sales trends and seasonal performance patterns,
 Determined high-revenue product categories and sub-categories,
 Compared profitability across customer segments,
 Evaluated business performance using Sales-to-Profit ratios,
 Highlighted areas with strong sales but lower profit margins.
